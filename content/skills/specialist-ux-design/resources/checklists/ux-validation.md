# Checklist de Validação de UX Design - Maestro Skills

## 🎯 Score mínimo de aprovação: 75 pontos

---

## 📋 Estrutura do Documento (15 pts)
- [ ] **Sumário executivo** completo
- [ ] **Arquitetura de informação** definida
- [ ] **Wireframes** listados e descritos
- [ ] **Design visual** documentado
- [ ] **Jornadas** mapeadas por persona
- [ ] **Prototipagem** com links válidos
- [ ] **Histórico de versões** atualizado

## 🧭 Arquitetura de Informação (10 pts)
- [ ] Mapa do site completo
- [ ] Fluxos principais descritos
- [ ] Navegação consistente
- [ ] Hierarquia visual clara
- [ ] Componentes globais definidos

## 🧱 Wireframes (20 pts)
- [ ] Lista de telas cobre 100% dos requisitos
- [ ] Componentes documentados
- [ ] Estados (default, loading, empty, erro) cobertos
- [ ] Responsividade mapeada (mobile/tablet/desktop)
- [ ] Anotações claras para desenvolvimento

## 🎨 Design Visual (10 pts)
- [ ] Paleta e tipografia definidas
- [ ] Iconografia e componentes padronizados
- [ ] Microcopy alinhado ao tom de voz
- [ ] Feedback visual para interações
- [ ] Considerações de acessibilidade (WCAG AA)

## 👣 Jornadas do Usuário (10 pts)
- [ ] Personas atualizadas e validadas
- [ ] Jornada principal detalhada por etapa
- [ ] Emoções e dores mapeadas
- [ ] Oportunidades identificadas por etapa
- [ ] KPIs vinculados à jornada

## 🧪 Prototipagem e Testes (10 pts)
- [ ] Protótipo navegável disponível
- [ ] Interações principais implementadas
- [ ] Teste com usuários realizado
- [ ] Feedback coletado e registrado
- [ ] Ajustes aplicados pós-teste

## 🔄 Integração com Requisitos (10 pts)
- [ ] Cada RF possui tela correspondente
- [ ] RNFs (performance, acessibilidade) respeitados
- [ ] Regras de negócio refletidas no design
- [ ] Restrições técnicas consideradas
- [ ] Interfaces externas representadas

## ✅ Checklist Final (10 pts)
- [ ] Tudo consolidado e entregue no arquivo `docs/03-ux-design/design-doc.md`
- [ ] Score ≥ 75 pontos

---

## 📊 Classificação
- **90-100 pts** → ✅ Excelente (pronto para Modelagem de Domínio)
- **80-89 pts** → ✅ Bom (ajustes menores)
- **75-79 pts** → ⚠️ Aceitável (revisões recomendadas)
- **< 75 pts** → ❌ Reprovado (revisar design)
