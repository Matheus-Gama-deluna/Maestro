# Guia - Mobile Design

## Design System Layers
1. **Tokens:** Cores, espaçamentos, tipografia
2. **Components:** Botões, inputs, cards
3. **Patterns:** Navegação, formulários, listas
4. **Templates:** Telas completas

## Acessibilidade Mobile
- Contrast ratio mínimo: 4.5:1 (texto normal)
- Touch targets: >= 44x44 pixels
- Screen reader support
- Dynamic type support

## Animation Principles
- **Duration:** 200-400ms para micro-interactions
- **Easing:** ease-in-out para natural feel
- **Performance:** Usar transform e opacity
- **Feedback:** Sempre dar feedback visual

## Platform Guidelines
- **iOS:** Human Interface Guidelines
- **Android:** Material Design
