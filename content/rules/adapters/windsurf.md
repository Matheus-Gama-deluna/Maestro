# Windsurf Adapter

Este adaptador formata as regras para o Windsurf IDE.

O conteúdo de `RULES.md` será inserido automaticamente pelo CLI.

**Paths IDE-specific:**
- Skills: `.windsurf/skills/`
- Workflows: `.windsurf/workflows/`
- Rules: `.windsurfrules`
