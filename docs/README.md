## Arquivo (_archive/)

Documentos historicos organizados em:
- **`root-legacy/`** - Planos e implementacoes anteriores que estavam na raiz do projeto
- **`old-docs/`** - Documentacao supersedida de versoes anteriores
- **`old-analysis/`**, **`old-plans/`**, **`old-tasks/`** - Analises e planos antigos
- **`backup-contents/`** - Conteudo antigo migrado para `content/`
